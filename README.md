# To-Do Web App

A simple and user-friendly To-Do Web Application built using HTML, CSS, and JavaScript. Users can add tasks with a date and time, mark tasks as completed, edit tasks, and delete tasks.

## Features

* Add new tasks
* Set due date and time
* Mark tasks as completed
* Edit existing tasks
* Delete tasks
* Simple and responsive design

## Project Structure

```
project-folder/
│
├── index.html

├── abhishek.js
└── README.md
```

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## How to Run

1. Download or clone the project.
2. Ensure all files are in the same folder:

   * index.html
   * abhishek.css
   * abhishek.js
3. Open `index.html` in any modern web browser.

## Usage

1. Enter a task in the task input field.
2. Select a date.
3. Select a time.
4. Click the **Add Task** button.
5. Use:

   * **Complete** to mark a task as done.
   * **Edit** to modify a task.
   * **Delete** to remove a task.
